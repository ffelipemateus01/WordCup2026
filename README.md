# ⚽ Copa do Mundo 2026

Um jogo de futebol 2D estilo *head soccer*, desenvolvido em **Python** com **Pygame**. Enfrente a CPU ou jogue contra um amigo no mesmo teclado, marque gols e dispute o topo do ranking de vitórias.

## ✨ Funcionalidades

- 🎮 **Dois modos de jogo:**
  - **Single** — jogador contra a CPU (bot)
  - **Versus** — dois jogadores no mesmo teclado
- 🏆 **Ranking de vitórias**
- 🎨 **Skins aleatórias**
- 🔊 **Trilha sonora e efeitos**

## 🛠 Tecnologias

- [Python 3.10+](https://www.python.org/)
- [Pygame](https://www.pygame.org/)
- [SQLite3](https://docs.python.org/3/library/sqlite3.html) (biblioteca padrão)
